# ai-world-cup-data-examples

Example structured football data files for the KnowNow AI World Cup Prediction Challenge.

This repository contains sample team dossiers, match context files and templates used to compare how humans, AI models, a betting market view and data-driven prediction methods respond to the same football fixtures.

The files are shared as examples of an AI football data analysis workflow. They show how structured background information, match-specific context and reusable Markdown templates can be used to support football score predictions with different levels of data.

## What is included

- Example team dossiers
- Example match context files
- Reusable dossier and match context templates

## Why this exists

The AI World Cup Prediction Challenge is a learning exercise rather than a definitive forecasting model. It is designed to explore how different models behave when given different levels of structured football data.

The examples in this repository are not intended to be a complete scouting database, betting model or professional forecasting system.
